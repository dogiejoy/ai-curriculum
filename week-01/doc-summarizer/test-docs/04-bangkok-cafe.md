# คาเฟ่ในกรุงเทพที่ทำงานได้

ในกรุงเทพมีคาเฟ่จำนวนมาก แต่ไม่ใช่ทุกที่ที่เหมาะกับการทำงาน วันนี้รวมเกณฑ์
ที่ควรพิจารณาก่อนเลือกคาเฟ่เป็น remote office

## เกณฑ์สำคัญ

**Wi-Fi เร็วและเสถียร** เป็นปัจจัยสำคัญที่สุด ลองทดสอบ speed test ในช่วงเวลาที่
ลูกค้าเยอะที่สุด (เสาร์-อาทิตย์บ่าย) ถ้ายังเร็วอยู่ ที่อื่นในเวลาปกติก็จะดีกว่า

**ปลั๊กไฟ** บางที่มีน้อยมากต้องแย่งกัน หลายแห่งจัดโต๊ะให้ติดผนังเพื่อให้เสียบปลั๊กง่าย
แต่บางที่กลับซ่อนปลั๊กไว้ลึกจนต้องลากสายยาว

**บรรยากาศและเสียงรบกวน** บางคนชอบบรรยากาศคึกคักช่วยให้มีโฟกัส บางคนชอบเงียบ
คาเฟ่ที่ใช้เพลงเปิดเสียงดังหรือมีเครื่องบดกาแฟตลอดเวลาอาจไม่เหมาะกับ video call

## ย่านที่แนะนำ

ทองหล่อ-เอกมัย มีคาเฟ่หลากหลาย แต่ราคาสูง อารีย์มีคอมมูนิตี้ remote worker
และร้านที่ออกแบบมาเพื่อทำงานชัดเจน บางซื่อ-วงศ์สว่างเริ่มมีคาเฟ่ดีๆ ที่ยังไม่แออัด

ลองสลับ 2-3 ที่ในแต่ละสัปดาห์ป้องกัน boredom + ได้ network ใหม่ๆ
